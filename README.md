## Installation

```sh
npm add css2object
```

## Usage

```js
import Css2Object from 'css2object';
const cssData = `
 h1 {
     color: blue;
 }

 body {
     color: red;
 }
`
const css2json = new Css2Object(cssData);
css2json.read();
console.log(css2json.cssRecord);
```
